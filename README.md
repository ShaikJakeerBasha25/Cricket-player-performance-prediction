# Cricket-player-performance-prediction
- This project is predict the next match Runs/Wickets for a player using ipl ball-by-ball dataset .
- 1.exploratory data analysis --for cleaning of data
- 2.feature engineering -( aggregation , rolling_avg , carrer avg , venue avg , etc..)
- 3.Model Training-using XGBRegressor by the metrics RMSE , MAE , R2 and done Hyper tuning , visualizing by SHAP
- 4.deployed Streamlit Dashboard for prediction
